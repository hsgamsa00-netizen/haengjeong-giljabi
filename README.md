# 화성 시민 행정 길잡이

화성 시민이 화성시 행정과 얽힌 상황에서 **받을 권리는 찾고, 절차는 알고, 부당하면 길을 안내받는** AI 행정 길잡이 웹앱.

**라이브**: https://hsgamsa00-netizen.github.io/haengjeong-giljabi/

## 이용 모드
- **시민·민원인**: 분야 선택 → ① 받을 권리 찾기(복지·지원금 자격 매칭) ② 처리 점검(법정 처리기한·구제 경로) ③ 부당함 점검(과태료 감경·이의 절차)
- **공직자**: 화성 자체감사 사례를 분야·키워드로 검색(착안점·원인·처분) — 업무 전 예방 점검

## 기술 구조
- **빌드타임 AI 구조화**: 법령·제도·감사사례를 시민용 체크포인트·자격 규칙으로 정제 → `data.js` (`build_data.py`)
- **런타임 규칙엔진**: 시민 입력에 자격 매칭·날짜계산·절차 검색을 결정론적으로 계산. **런타임 LLM 없음 · 입력정보 외부 전송 없음 · 오프라인 동작**
- 단일 페이지(`index.html`) + 데이터(`data.js`). 외부 의존성 0.

## 안내
공개 법령·제도·공공데이터를 바탕으로 한 **참고용 자가점검 안내**이며, 민원 접수·대리·법률자문이 아닙니다. 최종 자격·금액은 신청기관 심사로 확정됩니다. 데이터 기준일·출처는 화면에 표기됩니다.
